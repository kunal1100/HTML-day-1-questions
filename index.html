HTML Interview questions day 1

1. Difference between Div and Span

DIV = The div tag is known as Division tag. The div tag is used in HTML to make divisions of content in the web page like (text, images, header, footer, navigation bar, etc)

SPAN = The HTML span element is a generic inline container for inline elements and content. It is used to group elements for styling purposes (by using the class or id attributes)



2. Difference between Class and ID

Class = The class is an attribute which specifies one or more class names for an HTML element. The class attribute can be used on any HTML element. The class name can be used by CSS and JavaScript to perform certain tasks for elements with the specified class name.

ID = The id attribute is a unique identifier that is used to specify the document. It is used by CSS and JavaScript to perform a certain task for a unique element. In CSS, the id attribute is used using the # symbol followed by id.



3. What are self closing tags

Some HTML tags (like img and br ) don't have their own content. These are known as self closing tags or empty tags.

5. Difference between block and inline?

By default, inline elements do not force a new line to begin in the document flow. Block elements, on the other hand, typically cause a line break to occur (although, as usual, this can be changed using CSS).



6. Difference between em and strong?

Strong = The strong tag is used to define text with strong importance. The content inside is typically displayed in bold

Em = The em tag is used to define emphasized text. The content inside is typically displayed in italic
